Gradebook

The Gradebook lets you view and edit scores and assignment answers for one or more students.
  • Select the desired assignment(s). If you want All, select the All checkbox.

  • Select the desired students. If you want All, select the All checkbox.

  • Scores: Displays a summary table of the scores for the selected students and assignments. In addition, a detailed listing of all selected assignments is shown for each student selected.

  • Statistics: Provides summary statistics for an assignment. A plot of the distribution of scores as well as the number of students who gave each answer will be shown.

  • Grade: Grades assignments for the selected students. Normally, assignments are graded when they are received. However, if you change scores for the assignment, use the grade option to regrade the assignment.

  • Ungrade: Marks the assignment as not graded. This is useful if you change due dates on a test and want to let students change answers.

  • Histogram: Displays the score frequency distribution for the selected assignments. If you select all students and all assignments, the generated graph will be available to students when they click on the Histogram button. This button appears on the student menu only when a graph is available.

  • Data: Sends the raw data for the selected assignments to you via e-mail. The format will be username followed by the answers, separated by tabs. Each student's answers will be listed on a separate line. In the case of forecasts, students will have multiple lines.

    Example:
    Miller, George  3       5       1       tibia
    Johnson,Judy    2       1       1       fibula

    • Essay and long Blank questions will be displayed in the same format as other types questions, making them less easily read. You may wish to write your own programs to separate the essays from other questions.

  • Edit Assignment: Provides an opportunity to view student answers, edit essays and change the number of points available. You may only Edit one assignment at a time.

  • Delete: Deletes assignments student have submitted. Select the assignments and students and click on Delete. You will be shown a display and asked to select the assignments which you would like to delete.

  • Add: Adds student assignments. Select the assignments and student and click on Add. You will be shown a display and asked to select the assignments which you would like to add.
