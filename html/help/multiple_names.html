How to Select
Multiple Names in a List

If you click on a name in a listbox, you will select only that name. However, on some listboxes, depending on the type of computer you are using, you may select multiple names by following these instructions:
  • Macintosh:
    • To select consecutive items, hold the shift key and click on the first and last items you wish to include. The system will automatically add the items in between these two.
    • To select nonconsecutive items, hold down command key, and click on the items you wish to include.

  • UNIX (X-Windows):
    • Click on the desired items within the listbox.
  • Windows:
    • To select consecutive items, hold the shift key and click on the first and last items you wish to include. The system will automatically add the items in between these two.
    • To select nonconsecutive items, hold the control <ctrl> key, and click on the items you wish to include.
