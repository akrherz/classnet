How to Change
the Browser Background

Netscape Navigator users can select a background color which will over-ride all pre-defined backgrounds. Internet Explorer users can select a "default" background color, however this will display  only when there is no pre-defined background.

Windows:

  • Netscape Navigator
    • Select "Edit" from the Main Menu.
    • Under "Edit" select "Preferences".
    • Inside the "Preference Box", select "Colors" which is under "Appearance" in the column on the left.
    • Inside the box labelled "Colors", click on the color sample next to "Background".
    • A CMYK color picker appears. Use this to select the color of your backgrounds.
    • Once you have selected the color you desire, click on "ok".
    • This returns you to the "Preference Box". Click on the check box labelled, "Always use my colors, overriding page".
    • Click on "ok".

  • Internet Explorer
    • Select "View" from the Main Menu.
    • Under "View" menu, select "Internet Options" The "Internet Options" window will open.
    • Under "General" tab, click on the "Colors" button. (Near the bottom of the box.)
    • A "Colors" box will appear. UNcheck the box labelled "Use Windows colors".
    • Click on the color sample next to "Background".
    • A CMYK color picker appears. Use this to select the color of your backgrounds.
    • Once you have selected a color, click "ok".
    • Click "ok" again to close the "Colors" box.
    • Click "ok" again to close the "Internet Options" box.
