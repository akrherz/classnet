Adding a Class Member

  • Indicate whether you are adding a student or an instructor to the class.
  • Enter the individual's first and last name.
    • Please note: The combination of the first and last name must be unique for the class.
  • Enter the individual's e-mail address.
  • If you are adding an instructor, select privileges to:
    • Manage Students - Add, delete and edit student records.
    • Manage Assignments - Add, delete and edit assignments.

Press Back in your browser to return to the Add Member page.
