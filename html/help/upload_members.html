Uploading Student Data

ClassNet allows you to prepare a file of students you want to add or delete, and upload this information to the ClassNet server. This is useful when you have several student records to add or delete.


Preparing the file:

  • Open a text file.
  • The first line must read: #ClassNet1.0:Upload

  • List student information one student per line
    • To add a student, use the format:
      +lastname,firstname,password,email@address
    • Do not include any spaces.
    • To delete a student, use the format:
      -lastname,firstname
    • Do not include any spaces.
  • Comments need to be preceded by a pound sign (#).
    • #This is a comment.

  • You may wish to store the file in a WWW directory.
    • If possible, the file should be available only through a password.
    • It does not need to be an HTML file, viewable through a browser.

Example:

#ClassNet1.0:Upload
+Miller,George,apwd,gmiller@school.edu
#this is a comment
-Clooney,Mary

This file would add George Miller and delete Mary Clooney.

Uploading the File:

  • If the file is stored in a WWW directory, enter the URL for the file in the Remote URL field.
    • For example, http://www.school.edu/my_file
  • If the file is stored on your computer, click on Browse and locate the file.

  • Click on the Upload button.
    • If you do not click on Upload, your file will not be uploaded, and student records will not be modified.

     

Press Back in your browser to return to the Upload page.
