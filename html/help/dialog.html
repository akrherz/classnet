Creating a Dialog Assignment

A dialog assignment allows for private, ongoing conversations between instructors and students. This allows the instructor a means of organizing and storing conversations, and provides a means of communicating with students who do not have their own e-mail address.

The conversations are stored on the ClassNet server, and can be analyzed and synthesized at a later date. This assignment type is ideal for students in areas such as teacher education or counselor education, who value the opportunity to hold conversations with professionals and clients in their field of study. .

Creating the assignment:

  • Create a text file consisting only of HTML text and text areas.
  • For example:
    • A classroom teacher might create an image with a text area below it. He/she might ask students to write a story within the text area describing the picture.
    • A counselor may define various problem situations, each with an associated text area. He/she may have an ongoing private dialog with an individual student about each situation.
  • Please note that only form text areas are used and all HTML references must contain the full URL (e.g. src="http://www.public.iastate.edu/~mvg/images/image.gif")

 

Uploading the Assignment

  • Store the file in a WWW directory.

  • Add an assignment of the type Dialog.

  • Give the assignment a name.
  • Specify the URL for the file in the Initial URL field.
    • e.g. http://www.public.iastate.edu/~mvg/dialog.html

  • Click Add.

  • ClassNet will read the file and store it on the ClassNet server.

Click on Back in your browser to return to the Adding Assignments page.
