Students

If you are new to ClassNet:

  • Locate and select your class in the listbox.
  • Click on  Join a ClassNet Class.

To login to a class you have already joined:

  • Locate and select your class in the listbox.
  • Click on the Login button.

Note: Joining a ClassNet class does not formally register you in a university class. See the instructor or university registrar for questions concerning official class enrollment.

 


Instructors

To create a new class

To login to an existing class

  • Click on the class name in the listbox
  • Click on the Login button
