Creating an In-Class Assignment

An in-class assignment allows you to enter scores for tests and assignments completed outside of ClassNet. Once entered, these scores will be reported along with all other scores managed by the ClassNet server.

 

Create an in-class assignment:

  • Add a new assignment.
  • Assign a name.
  • Designate the type as INCLASS.

 

Entering Scores From a Text File:

  • The file must be in this format:
    • Last First  ....other information...  Score
    • ClassNet reads only the student names (first and last) and the assignment score. All other information is ignored.

  • Put the file in a Web directory.

  • Specify the URL for the text file when you create the in-class assignment.
    • Example: http://www.public.iastate.edu/~cs403/exam1.dat

  • ClassNet will read the file, and provide a report showing the score each student was assigned, a list of students who received no score, and a list of lines in your data file which could not be matched to a student name. In the latter two cases, you can manually enter the necessary scores.

  • Submit the report to save the scores on the server.

  • Remove the data file from the Web directory to protect the confidentiality of the information.
    • Note: It is possible to create a Web directory which can only be accessed through a password. See your local administrator for details and assistance.

  • Edit the assignment as necessary.
    • You may change the maximum score for the assignment.

  • Click on the Publish checkbox to make scores visible to students. Be sure to save your changes.

Data File Example: 

Jones Mary J.   111111111 35
Miller George   030200202 22

 

Changing Student Scores

  • Click on Gradebook in the Instructor menu.

  • Select the in-class assignment in the Assignment listbox.

  • Locate and click on the students whose scores you wish to change. If you wish to change the scores for all students in the class, click on All below the student listbox.

  • Click on Edit Assignment.
You will be given a list of scores and the accompanying student names. Enter the new score for each student and click on Submit to save the changes. The score you enter must be between 0 and the total number of points possible for the assignment.

Click on Back in your browser to return to the Editing a Test page.
