Editing Basics

  • Add a new block: Select the assignment name and click Add. A new block
    will be added at the end of the test.

  • Add a new question: Select the block in which the new question should be added,
    and click Add. A new question will be added at the end of the block. Alternatively,
    select a question and click Add to add a new question after the selected question.

  • Delete a section: Select the desired section and click Cut. The section will be removed and copied to a clipboard (See also Paste).
  • Edit a Section: Select the desired section and click Edit. An editor will appear in the
    frame to the right.

  • Copy a section: Select the desired section and click Copy. The section will be copied to the clipboard.

  • Paste a section: Select the section where you want to Paste your data.
    • If the section is a block, the last deleted or copied block will be pasted.
    • If the selected section is a question, the last deleted or copied question will be pasted. If a section wasn't previously deleted or copied, a new section will be pasted.

  • Move a section: Cut it from the old location and paste it to the new location.
  • View your test: Click on View.
Click on Back in your browser to return to the Editing a Test page.
