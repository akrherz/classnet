Student Evaluation

Presenting information via the Web, whether it be text, audio, video or other media is only part of the instructional process.  Students also need to be given assignments and evaluated on their performance.  A great deal of attention has been paid to information delivery.  Much less attention has been devoted to developing ways to evaluate students.  Furthermore, assessment exercises should also be learning tools.  See the National Science Education Standards and the Assessment chapter.

This page is intended to present some existing alternatives we have used to present and evaluate assignments on the Web.  Since evaluation on the Web is in its infancy, please view the examples as suggestive rather than exhaustive.

As we look at different evaluation instruments, we need to be aware of what that instrument can realistically evaluate.  While multiple-choice tests are a mainstay of many courses and are easy to construct and grade, their ability to evaluate higher levels of thinking is low.  Below is a table of some evaluation types and thinking levels (from Bloom's taxonomy)

Some have said you get what you test.  Would you agree?

Education has been criticized for producing students who know isolated facts but can't apply their knowledge to realistic situations.  Can we use the Web to remedy that situation?

At Iowa State, we have several tools which can help support the evaluation types described above.  We hope that these examples motivate you to come up with new types which will foster higher levels of thinking  for students.

Global Change Course

The Global Change Course at Iowa State as implemented an electronic portfolio for evaluating student progress. Defined, a portfolio is a "representative and judicious collection of the student's work." The student portfolio for this course has two fundamental purposes:

  1. Provides a documentation of the student's work, and
  2. Serves as the basis for evaluating the student's work based upon given standards.

The benefits of the portfolio include:

  1. Providing the student with criteria to be used in judging their work.
  2. Providing the student with direct evidence of their work.
  3. Providing the student with a chance for self-analysis and reflection.
  4. Serving as a form of "electronic publishing."

You are welcome to explore some of the features of the Global Change Course Electronic Portfolio which include both student and instructor versions. Matthew T. Johnson, a recent graduate and student in the course, has graciously offered his portfolio for you to examine.

Class Management

Managing assignments and scores gets more complicated as we move to an Internet-based education system.  Students may no longer be in the same country, let alone the same campus.  Time zones and student schedules make synchronous delivery of information less viable.

The ClassNet system was designed to support the administrative load of Internet classes. Using ClassNet, you can create a class which students enroll themselves in, create assignments (tests, simulations, problem sets etc.) which students complete, score the tests often automatically and make results available any time, anywhere.

The best way to understand ClassNet is to use it. Here's how:

  1. Go to ClassNet
  2. Click on Enroll as a Student on the Main Menu
  3. Specify the class name Education Workshop and give the requested information.
  4. Click on Enroll
  5. Login to the class
  6. Complete each of the assignments listed:
  • A Problem Set - This is an example of a self-test consisting of a number of different types of questions.  Click on A Problem Set and Complete.
    1. Fill in the answers and Submit the test.
    2. Click on Answers.  How did you do?
    3. Click on Back and then Scores.  Students can view their current scores at any time from anywhere.
  • Forecasting - Complete the forecast.  Note that a forecast is like the problem set in which the answers change daily.  Come back tomorrow to see how you did.

    Mountain - Assignments can be anything which can be shown on a Web page.  Try out this Java simulation.  We can build in the capabilities to save the student attempts and replay them.
