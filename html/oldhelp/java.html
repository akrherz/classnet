Java Help

A java assignment is a more complicated assignment that allows one to store java simulation data. This allows instructors and researchers to analyze how students are working with simulations. This type of an assignment involves designing a java applet which interacts with a second data server running on ClassNet's end. This assignment is proving to be a powerful assignment type for simulation research and needs to be developed in consultation with ClassNet administrators.

 However, one can create a simple java assignment that does not store data in ClassNet and does not involve consultation with ClassNet administrators. By following the directions below, one can give students private access to an assignment inside ClassNet containing a Java applet.

Assignment Creation

Using your favorite HTML editor, create an assignment file consisting of html text and a java applet.

Placing the Assignment into Classnet

  1. Put the html file and java class(es) into a Web directory.
  2. Go to the Instructor Assignments menu, and click Add.
  3. Select Type: Java and give it a name. This name must be the same name as the html file without the html extension. For example if the html file in the first step is called java1.html, then the assignment must be named java1. Currently, spaces are not allowed in java assignment names.

    4. For Initial URL, specify the Web directory where you put the assignment in the first step above.
    (e.g. http://www.public.iastate.edu/~mvg/java1.html)
  4. Click Add, and ClassNet will read this file and attempt to store it within ClassNet. If the assignment was stored correctly, you will be presented with an updated Assignments Menu.
  5. To load the java applet classes, again click on Add in the Assignments Menu. Select Type: Java and give it the assignment name used (e.g. java1). Then, in Initial URL, specify where the applet class is. (e.g. http://www.public.iastate.edu/~mvg/java1.class).

    6. Repeat this step for additional classes.
  6. If you wish to make this assignment immediately available to the students, select the assignment name and click Edit in the Assignments Menu. In the form you receive, click the Publish checkbox and then press Save.
Notes
Even storing a simple java assignment within ClassNet is a more detailed process than the other assignment types. Please contact ClassNet administrators with any questions.
One of the quickest ways to store the classes is to create a single archive by zipping the classes with no compression into one .zip file.
An example session
LessonGraph is an example java assignment html file that uses a zip file. Note that LinearLesson.class is actually stored within LessonGraph.zip. After loading LessonGraph.html, one would only need to add in LessonGraph.zip

 Example Steps I used

  1. I put the LessonGraph.zip and LessonGraph.html files into my web readable directory at Iowa State. This directory is accessible by the URL http://www.public.iastate.edu/~mvg
  2. Logged into my class as Instructor and went to the Assignments menu where I clicked on Add
  3. Type: Java

    4. Name: LessonGraph
    Initial URL: http://www.public.iastate.edu/~mvg/LessonGraph.html
    Clicked Add
  4. Clicked on Add in the Assignments Menu
  5. Type: Java

    6. Name: LessonGraph
    Initial URL: http://www.public.iastate.edu/~mvg/LessonGraph.zip
    Clicked Add
  6. Selected "LessonGraph" Clicked on Edit in the Assignments Menu
  7. Clicked the Publish checkbox and clicked on Save
The assignment was now available to my students.
