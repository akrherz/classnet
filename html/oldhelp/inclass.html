Inclass Help

An inclass assignment allows you to enter scores for student from inclass tests or other assignments. These scores will be reported along with other scores managed by ClassNet.

To create an inclass assignment, add an assignment, name it and make it an INCLASS assignment.

Initial URL

You can read scores from a file. Here's how:
  1. The file should consist of lines in the following format:
Last First  ....other information...  Score
    Example:
Jones Mary J.   111111111 35
Miller George   030200202 22
ClassNet only uses the first name, last name and the score, ignoring other information.
  1. Put the file in a Web directory.
  2. When you add the assignment, specify this file name in the Initial URL field. Example: http://www.public.iastate.edu/~cs403/exam1.dat
  3. ClassNet will read this file and respond with a form consisting of the score each student received on the test, a list of students who didn't receive a score and a list of lines in your file which could not be matched with a student name. In the latter two cases, you can manually enter the scores. You can also set the maximum number of points for the test.
  4. Submit this form to store the scores.
  5. Move the file out of the Web directory to keep others from reading it.
Note
It is possible to create a Web directory which is password-protected. This would be a good place to store your files. See your local Webmaster for details.

To make the scores visible to students, edit the assignment and click on the Publish checkbox and Save. You may also change the maximum score for the assignment.

Changing Scores

To change student scores for the inclass assignment:
  • Click on Gradebook in the Instructor menu.
  • Select the inclass assignment in the Assignment listbox
  • Select the students you want to give scores to or click on All below the student listbox for all students.
  • Click on Edit Assignment.
You will be shown a list of scores and student names. Enter the score for each student and click on Submit to make the changes. The score you enter must be between 0 and the total number of points possible for the assignment.
