Assignment Help

To edit or delete an assignment:
  1. Select the desired name in the assignment list
  2. Click on Edit or Delete. If you choose to edit, an editor appropriate to the assignment will be displayed.
To add an assignment, Click on Add. Supply the name (limit 15 characters) and the type of the assignment.

Click on Mail to send a copy of the Assignment to your mailbox. The copy is useful for archiving or for modification using a text editor. Later you can upload this file to rebuild your assignment or load it into a new class you have created.

Dr. Carol Chapelle has written an excellent introduction to making tests.

Help on editing tests is also available here and in the test editor.

Assignment Types

ClassNet is designed to support a growing list of assignment types. For example, a TEST is a multi-question test consisting of multiple-choice, fill-in-the-blank or essay questions. A FORECAST is a test which requests tomorrow's weather data for a site in the U.S.

When students submit this test, ClassNet will grade it according to the answers you have given. Future types of assignments may be SURVEYS (i.e. ungraded tests), PORTFOLIO entries (i.e. files created by students) or even the output of programs. If you have suggestions for assignments which meet your needs, let us know.

Publishing

When an assignment is initially created, it is unpublished which means that students will not see it on their assignment menu. Using the Edit option, you can supply the necessary information for the assignment. When it is ready for use, select the Publish option to make it available to students.
