Edit Test Help

Editing Assignments Blocks Questions

A test consists of a list of blocks. Within each block is a list of questions. Blocks and questions are called sections. The Test editor will allow you to add, delete and edit assignment, block and question values.

Once you have published an assignment, students can complete it. When they request to complete an assignment, ClassNet will read all your assignment information and generate a test.
Students will then provide the answers and submit the test to ClassNet for grading.

Editing

  • To add a new block - select the assignment name and click on Add. A new block
    will be added at the end of the test.
  • To add a new question - select the block in which the new question should be added
    and click on Add. A new question will be added at the end of the block. Alternatively,
    select a question and click on Add to add a new question after the selected question.
  • To delete a section - select the desired section and click on Cut. The section will be removed and copied to a clipboard (see paste to learn how to retrieve it).
    To edit a section - select the desired section and click on Edit. An editor will appear in the
    frame to the right.
  • To copy a section- select the desired section and click on Copy. The section
    will be copied to the clipboard.
  • To paste a section - select a section where you want to Paste. If the section is a block, the last deleted or copied block will be pasted. If the selected section is a question,
    the last deleted or copied question will be pasted. If a section wasn't previously deleted or copied, a new section will be pasted.
  • To move a section - cut it from the old location and paste it to the new location.
  • To see a sample test - click on View

Assignments

Editing an Assignment


Click on the assignment name (the first name in the list) and click on edit in the left frame. In a few seconds, your editing options for the assignment will appear at the right:

Question Selection - you can choose to randomly select from the questions in each block or randomly select a version of the test. ClassNet creates versions by selecting all the first questions, all the second questions etc.

Student Answer Options

You may require that student answer all questions. Requiring answers is appropriate for a test but may not be for a survey. When a student submits a test which requires answers, he or she will be asked to supply all answers before the test is accepted for grading.

You may also allow students to see the answers to the test along with their answers after the due date.

Due Date

If you don't specify a due date, the assignment will be accepted at any time. If you do, no assignments will be accepted after the due date. You can supply just the date (e.g. 09/11/1996
which is September 11, 1996) or you can include the hour and minute (e.g. 13:30:09/11/1996 \
which is 1:30PM on the same day). Note that the hour is based on a 24 hour clock.

Proctor Password

If you specify a proctor password, the student will not be able to complete this assignment until the password is entered. When the student clicks on the Complete button in the Student menu, the password will be requested. If it is correct, the assignment will be displayed. If no password is supplied, the assignment will be displayed immediately.

Publish


While you are editing an assignment it remains inaccessible to students. Once it is published, students can complete it. To publish or unpublish an assignment, select the appropriate value in the checkbox and Save.

Blocks

Give your blocks descriptive names to distinguish them in the editor. The block name is not seen
by the student.

You can specify the total points for this block (default 1). Total points apply to any question presented in the block and are awarded if the answer is correct. The total points on the test are the sum of the total points of the blocks.You may also specify partial points for students completing the question even if it is wrong.
Prior to the presentation of the question in the block, you may preface the question with any other text you desire by typing it in the preface. This text may include HTML tags which will allow you to present graphics, video, audio, music or even simulations in your questions.

Questions

Your question can be one of several types:
  • CHOICE - a typical multiple-choice question. The student can supply one of several answers. To create, select the CHOICE type and enter your question text. Before each choice, enter the <?> symbol. When the student sees the question, a radio button will replace this symbol. For the answer, specify the number of the choice which is correct. ClassNet will automatically grade these answers.

    Example 
Which is a type of question?
<BR><?>CHOICE
<BR><?>LISTING
<BR><?>NOTHING

Note

The <BR> symbol is HTML for a line break and causes each answer to appear on
a new line. Without the symbol, the answers would appear on the same line. You may intersperse HTML commands in your text to control the format. All questions
will begin with a new paragraph and end with a horizontal rule.

  • BLANK - a typical fill-in-the-blank question. Select the BLANK type and enter your question text, delimiting your blank by <?>.....</?> delimiters (one line only). The blank will be the size of the blank you specify and any text between the delimiters will be the initial value of the blank. You may have multiple blanks.


    Example 
The type of question which requires a student
answer is <?>name...</?>.
    For the answer, type it in the answer editor and specify the judging options below as needed. Separate answers by commas for multiple blanks. Alternate answers are separated using the '|' symbol. ClassNet will automatically grade these answers.

    Example

    choice | option, blank | essay

    This would provide the answers for two blanks. The answer for the first blank would
    be either choice or option. The answer for the second blank would be blank or essay.

  • ESSAY - a text editor in which students can enter free-form answers. Select the ESSAY type and type in your text. Delimit the text editor using <?>....</?> . These may extend over multiple lines. Any text between the delimiters will be the initial value of the editor. Specify the size of the editor in the Essay Editor Size fields. The answer must be manually graded.

    Example 
What is your opinion of this software:
<?>
I believe that the software is......

</?>
  • OPTION - a popup list of option names. Select the OPTION type and type in your text, placing a <?> before each option and a </?> after the last option. Put the text for the correct option in the answer. The answer will be automatically graded.

    Example 
Choose a type of question:
<?>BLANK
<?>ESSAY
<?>OPTION
</?>

  • LIST - a listbox of options. Select the LIST type and type in your text, placing a <?> before each option and a </?> after the last options. Put the text for the correct option in the answer. The answer will be automatically graded.

    Example 
Choose a type of question:
<?>BLANK
<?>ESSAY
<?>OPTION
</?>
  • MULTIPLE - a multiple-choice question which allows more than one correct answer. Select the MULTIPLE type and enter your question text. Before each choice, enter the <?> symbol. When the student sees the question, a checkbox will replace the symbol.

    Example 
Which is a type of question?
<?>CHOICE <?>LISTING <?>NOTHING
  • LIKERT - a multiple-choice question which generally represents a scale from one extreme to another. Likert scales are specified like CHOICE questions. However, since there is no
    right answer, a score is generated based on the answer selected. By default, the score is based on the order of the answer chosen. The first answer would be scored 1, the second 2, etc. Weights can be specified in the answer field if another scoring scheme is desired. The
    weights should be numbers (plus or minus) separated by commas.

    Example
    Answer:     5,4,3,2,1 
Your view of computers is:
<?>Very unfavorable <?>Unfavorable <?>Neutral  <?>Favorable <?>Very Favorable
The above example would present a Likert scale of five items which are reversed scored.


Feedback


You can supply feedback for this question which will be displayed when the student views the answer (if you allow it). This would be a good place to put links to relevant information for
this question.

Blank Judging Options


To grade a Blank question, ClassNet compares the correct answer with the student answer. You can control this judging process by using the following options:
  • Numeric - treat the answer as a number and compare it numerically. Your answer can be a value like 3.14 or a range like -5.00:10.33. If Numeric is selected, all of the following options are ignored.
  • Textual - a text comparison will be made. This comparison is controlled by the following options:
  • Exact - the comparison is letter for letter. All the remaining judging options are ignored. If not selected, the following options are enforced if selected:
  • Capitalization - 'a' and 'A' will be considered different
  • Spelling - words must be spelled exactly as in the answer. Otherwise, words that are "close" are accepted as the same.
  • Punctuation - enforce the same punctuation as in the answer. Normally, punctuation is ignored.
  • Word Order - if the order of words in the answer is important, select this option.
  • Word Count - if you are only looking for a specified number of words in the answer select this option. Otherwise, extra words will be allowed.

Viewing


The best way to learn how to build a question is to enter text in the editors and click on View.
The question as it appears to the student will appear in a separate window. Your data won't be saved until you click on Save.
