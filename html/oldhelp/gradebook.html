Gradebook Help

The Gradebook lets you view or edit assignment answers or scores for one or more students.
  1. Select the desired assignments (if you want All, select the All checkbox)
  2. Select the desired students (if you want All, select the All checkbox)
  • Scores displays a summary table of the scores for the selected students and assignments. In addition, a detailed listing of all selected assignments is shown for each student selected.
  • Statistics provides summary statistics for an assignment. A plot of the distribution of scores as well as the number of students who gave each answer will be shown.
  • Grade regrades assignments for the selected students. Normally,assignments are graded when they are received. However, if you change points for the assignment, use the grade option to regrade the assignment.
  • Ungrade will mark the assignment as not graded. This is sometimes necessary if you change due dates on a test and want to let students change answers.
  • Histogram will display the score frequency distribution for the selected assignments. If you select all students and all assignments, the generated graph will be available to students when they click on the Histogram button. This button only appears for the students if the graph is available.
  • Data sends the raw data for the selected assignments to you via email. The format will be username followed by the answers, separated by tabs. Each student's answers will be listed on a separate line. In the case of forecasts, students will have multiple lines. For example:
Miller, George  3       5       1       fibia
Johnson,Judy    2       1       1       fibula
    ESSAY or long BLANK questions will be displayed like other questions, making the format less readable. You may need to write your own programs to separate the essays from other questions.
  • Edit Assignment lets you view student answers, edit essays and change the number of points. You may only Edit one student assignment at a time.
  • Delete lets you delete assignments student have submitted. Select the assignments and students and click on Delete. You will be shown a display and asked to select the assignments which you would like to delete.
  • Add lets you add assignments which students have not submitted. Select the assignmnets and student and click on Add. You will be shown a display and asked to select the assignments which you would like to add.
