Main Menu Help

Students

To login to a ClassNet class you have already joined
  • click on the class name in the listbox
  • click on the Login button.
If you are new to ClassNet, click on  Join a ClassNet Class.
 
Note
Joining a ClassNet class does not formally register you in a university class. See the instructor or university registar for questions concerning official class enrollment.

Instructors

To login to a ClassNet class you have already created or joined
  • click on the class name in the listbox
  • click on the Login button
To create a class, click on Create a ClassNet Class.
